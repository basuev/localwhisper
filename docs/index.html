Features

Private by design

All transcription runs locally on your Mac. Your voice data never leaves your device.

Apple Silicon native

Built on MLX Whisper, optimized for M-series chips. Fast, efficient, no GPU bottleneck.

opt

One-hotkey workflow

Press Option to record, release to transcribe. The result pastes into your active app automatically.

10+ languages

Russian, English, German, French, Spanish, Japanese, Chinese, Korean, Ukrainian, Polish.

LLM post-processing

Automatic punctuation and grammar correction via Ollama locally or OpenAI in the cloud.

Streaming transcription

Processes audio in 5-second chunks during recording for faster results on long dictations.

Installation

Requires macOS on Apple Silicon (M1+), Python 3.11+, and uv.

$ git clone https://github.com/basuev/localwhisper.git
$ cd localwhisper
$ bash scripts/install.sh

After installation, grant Accessibility permission in System Settings.
